# 📖 About

This repository documents my learning journey with **Spring Boot**, covering concepts from backend fundamentals to building scalable RESTful APIs.

Rather than focusing on a single application, this repository is organized as a structured collection of notes, examples, and mini projects.

The objective is to strengthen backend development skills while creating a reusable knowledge base for future projects.

---

# 🎯 Learning Goals

* Master Spring Boot Fundamentals
* Understand Spring Core
* Build REST APIs
* Work with MySQL using Spring Data JPA
* Learn Dependency Injection
* Implement Authentication & Authorization
* Apply Clean Architecture principles
* Prepare for Backend Developer Internship
